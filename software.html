Tools & Software

🧠

LASHiS — Longitudinal Automatic Segmentation of Hippocampal Subfields

A multi-contrast MRI pipeline for the longitudinal automatic segmentation of hippocampal subfields. Uses ANTs-based registration and multi-contrast input to improve segmentation reliability across timepoints, particularly at ultra-high field.

Python ANTs 7T MRI

GitHub · Paper (NeuroImage 2020)

🧬

TongueSegMRI — Tongue Musculature Segmentation

A field guide and validation framework for segmenting the human tongue musculature from MRI, developed and validated in a cohort of people living with Motor Neuron Disease. Includes multi-site, multi-contrast annotated datasets.

Python Segmentation MND

GitHub · Paper (CIBM 2025)

📈

Neurodesk — Reproducible Neuroimaging Environment

An accessible, flexible and portable data analysis environment for reproducible neuroimaging. Packages hundreds of neuroimaging tools into containers, enabling consistent analysis across platforms and institutions worldwide.

Containers Open Science Nature Methods

neurodesk.org · Contributors · Paper (Nature Methods 2024)

🧬

VesselBoost — Blood Vessel Segmentation

A Python toolbox for small blood vessel segmentation in human MR angiography data. Uses deep learning to improve vessel detection in high-resolution MRI datasets.

Python Deep Learning MR Angiography

GitHub · Paper (Aperture Neuro 2024)

🧠

OSHy-X — Hypothalamic-ForniX Atlas & Segmentation

Open-source atlases and segmentation tool for the hypothalamus and fornix at 3T and 7T. Enables automated segmentation of subcortical structures relevant to MND and metabolic research.

Python JOSS MND

GitHub · Paper (JOSS 2022)

🏹

BodyLocaliser — fMRI Body Localiser Task

A body localiser experiment for fMRI built using PsychoPy. Designed for mapping somatotopic representations of the body in the motor cortex, supporting MND research into cortical body mapping.

Python PsychoPy fMRI

GitHub

🎬

DICOM2Vid — DICOM to Video Converter

A lightweight tool to convert DICOM image folders into video files, useful for sharing MRI data with participants and clinical staff in an accessible format.

Python DICOM Utility

GitHub

Datasets

TOMCAT Dataset

Towards Optimising MRI Characterisation of Tissue — a comprehensive dataset including all LASHiS longitudinal hippocampal subfield segmentation data. Openly available for use by the research community.

Data in Brief (2020)

Multi-site Tongue MRI Dataset

An annotated multi-site and multi-contrast MRI dataset for the study of the human tongue musculature, collected across multiple Australian centres and shared openly via Scientific Data.

Scientific Data (2025)

MND fMRI Appetite Dataset

An fMRI dataset for appetite neural correlates in people living with Motor Neuron Disease, shared openly to support research into metabolic dysfunction and non-motor symptoms in ALS/MND.

Scientific Data (2025)
